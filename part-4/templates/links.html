URL Building with url_for()

Instead of hardcoding URLs, Flask provides url_for() to generate URLs dynamically!

Why use url_for()?
  • URLs update automatically
  • Safer than hardcoding
  • Easy navigation
  • Works in Python & templates

Generated URLs

url_for() Call Generated URL Test
url_for('home') {{ links.home }} Visit
url_for('about') {{ links.about }} Visit
url_for('user_profile', username='Alice') {{ links.user_alice }} Visit
url_for('show_post', post_id=1) {{ links.post_1 }} Visit
url_for('category_product', category_name='electronics', product_id=1) {{ links['Electronics - Mobile'] }} Visit
url_for('category_product', category_name='electronics', product_id=2) {{ links['Electronics - Laptop'] }} Visit
